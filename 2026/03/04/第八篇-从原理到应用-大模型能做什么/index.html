复习：大模型的训练

上一篇我们详细讲了大模型的训练三阶段：

  1. 预训练：学习语言和世界知识
  2. 有监督微调（SFT）：学会遵循指令
  3. 强化学习对齐（RLHF）：对齐人类价值观

训练完成后，大模型能做什么？它有什么局限？

这就是今天要讲的内容。

大模型的本质

在讲应用之前，我们先理解大模型的本质：

大模型是一个超级大的”下一个词预测器”。

当你输入”今天天气”，它计算的是：

在训练数据里，”今天天气”后面最可能出现什么词？

然后选概率最高的那个，接着再算下一个词……直到生成完整的回答。

这解释了为什么大模型能”创作”

它不是在”思考”，而是在”预测”。

但神奇的是：

  • 当预测能力足够强，它看起来就像”理解”了
  • 当训练数据足够多，它”记住”了海量知识
  • 当模型足够大，它涌现出了”推理”能力

能做什么：

  • 回答各种问题（知识、生活、学习）
  • 进行多轮对话，记住上下文
  • 用不同的语气、风格回答

  • 写文章、博客、小说
  • 写诗歌、歌词
  • 写邮件、报告
  • 头脑风暴、提供创意

3. 编程辅助

代表产品： GitHub Copilot、Cursor

能做什么：

  • 代码补全
  • 根据注释生成代码
  • 解释代码功能
  • 找出代码bug
  • 代码重构建议

4. 翻译

能做什么：

  • 多语言互译
  • 保持原文风格和语气
  • 翻译专业术语

5. 文本摘要

能做什么：

  • 长文摘要
  • 提取关键信息
  • 生成不同长度的摘要

6. 教育辅导

能做什么：

  • 解释概念
  • 解答习题
  • 制定学习计划
  • 模拟考试

7. 多模态应用

代表产品： GPT-4V、Gemini、DALL-E

能做什么：

  • 看图说话：描述图片内容
  • 图像理解：分析图表、解答数学题（看图）
  • 图像生成：根据文字描述生成图片

大模型的局限

1. 幻觉（Hallucination）

大模型会一本正经地胡说八道：

  • 编造不存在的论文和作者
  • 给出错误的数学答案
  • 混淆事实

为什么会幻觉？

因为大模型不是在”知道”，而是在”预测”。它预测的是”这句话听起来像真的”，而不是”这是真的”。

2. 知识截止

大模型的知识来自训练数据，而训练数据有截止日期。

  • GPT-4的知识截止到2023年底
  • 之后发生的事情它不知道

3. 没有真正的理解

大模型不理解它说的话的含义。它只是根据统计规律，生成”最可能”的文本。

4. 推理能力有限

虽然大模型有一定的推理能力，但在复杂推理任务上仍会犯错。

5. 偏见和有害内容

训练数据来自互联网，包含了人类的偏见。

如何正确使用大模型

✅ 应该做的

  1. 验证重要信息：对关键事实，要交叉验证
  2. 提供上下文：给模型足够的背景信息
  3. 明确指令：告诉模型你想要什么格式、风格
  4. 迭代优化：第一次回答不满意，继续追问、调整

❌ 不应该做的

  1. 盲目相信：不要把大模型的输出当成绝对真理
  2. 用于高风险决策：医疗、法律、金融等，需要专业判断
  3. 输入敏感信息：你的数据可能被用于训练
